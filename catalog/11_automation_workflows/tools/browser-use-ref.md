## Relevance to Automation & Workflows
Browser Use (85.4K stars) is the leading open-source AI browser automation framework. While cataloged under AI Agents, it is equally important for the Automation category because:

- **Browser automation**: Autonomous web browsing, form filling, data extraction
- **Workflow integration**: 1,000+ integrations (Gmail, Slack, Notion)
- **Task automation**: Automates multi-step web-based business processes
- **AI-powered**: LLM reasoning makes automation resilient to UI changes
