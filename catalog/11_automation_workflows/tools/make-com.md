## What It Does
Make.com (rebranded from Integromat in 2022) is a visual automation platform that allows users to build complex multi-step workflows ("scenarios") by connecting apps and services. It differentiates from Zapier through its more powerful visual editor, better handling of complex data transformations, and more affordable pricing.

Make.com has added AI capabilities including an AI assistant for building scenarios, OpenAI/LLM integration modules, and AI-powered data mapping. Its visual approach makes complex logic (branching, loops, error handling, aggregation) more intuitive than Zapier's linear model.

## Key Features
- **Visual scenario builder**: Flowchart-style automation editor
- **1,500+ integrations**: Popular apps, databases, APIs
- **Advanced data transformation**: Array manipulation, JSON parsing, text functions
- **AI assistant**: Describe scenarios in natural language
- **OpenAI/LLM modules**: Built-in AI actions for text processing
- **Routers**: Split flows into parallel branches
- **Iterators/Aggregators**: Loop and collect data within flows
- **Error handling**: Try/catch-style error management
- **Webhooks**: Custom HTTP triggers/actions
- **Data stores**: Built-in simple database tables
- **Scheduling**: Cron-like execution schedules
