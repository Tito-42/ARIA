## What It Does
Agentic Security fournit des tests de sécurité spécialisés pour les agents IA et les applications LLM, avec une interface web pour visualiser les résultats. Spécialisé dans la sécurité des systèmes agentiques.

## Key Features
- **Tests agents IA**: Spécialisé systèmes agentiques
- **Interface web**: Visualisation des résultats
- **Multi-attack**: Différents types d'attaques
