## What It Does
L'OWASP Top 10 for LLMs est le document de référence standardisant les 10 risques de sécurité majeurs pour les applications basées sur des LLMs. Il sert de guide pour les équipes de développement, de sécurité et de compliance.

## Key Features
- **10 risques standardisés**: Prompt injection, data leakage, hallucination, etc.
- **OWASP**: Standard industrie reconnu
- **Mis à jour régulièrement**: Adapté aux nouvelles menaces
- **Guidance**: Recommandations de mitigation par risque
- **Communauté**: Contribution de centaines d'experts

## Top 10 Risks (2025 Version)
1. **LLM01** - Prompt Injection
2. **LLM02** - Sensitive Information Disclosure
3. **LLM03** - Supply Chain Vulnerabilities
4. **LLM04** - Data and Model Poisoning
5. **LLM05** - Improper Output Handling
6. **LLM06** - Excessive Agency
7. **LLM07** - System Prompt Leakage
8. **LLM08** - Vector and Embedding Weaknesses
9. **LLM09** - Misinformation
10. **LLM10** - Unbounded Consumption
