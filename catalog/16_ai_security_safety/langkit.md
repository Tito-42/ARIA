## What It Does
LangKit fournit des métriques d'observabilité pour les applications LLM : qualité des réponses, toxicité, sentiment, PII, et plus. Intégré avec WhyLabs pour le monitoring en production.

## Key Features
- **Métriques LLM**: Toxicité, sentiment, PII, qualité
- **WhyLabs integration**: Dashboard monitoring
- **Apache 2.0**: Licence permissive
