## What It Does
Stable Audio Tools fournit le code de training et d'inference pour les modèles de génération audio de Stability AI. Permet de générer de la musique et des effets sonores à partir de descriptions textuelles, avec la possibilité d'entraîner des modèles custom.

## Key Features
- **Text-to-audio**: Génération à partir de descriptions
- **Training code**: Entraîner ses propres modèles
- **MIT licence**: Code ouvert
- **Stability-backed**: Recherche Stability AI
