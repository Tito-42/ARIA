## What It Does
Zed est un éditeur de code haute performance construit en Rust par les créateurs d'Atom et Tree-sitter. Il combine performances natives, collaboration multiplayer temps réel, et intégration IA. Cross-platform (macOS, Linux, Windows), 36,592+ commits.

## Key Features
- **Performance Rust-native** : démarrage et édition ultra rapides
- **Multiplayer** : collaboration temps réel
- **IA intégrée** : agent capabilities, Copilot support, multi-model
- **Code review** : suggestions assistées par IA
- **Extensions** : écosystème d'extensions
- **36,592+ commits** : développement très actif
- **Cross-platform** : macOS, Linux, Windows

## Strengths
- Le plus rapide des éditeurs modernes (Rust natif)
- Collaboration multiplayer native (pas un add-on)

## Limitations
- Écosystème extensions plus jeune que VS Code
- IA moins intégrée que Cursor
- Migration depuis VS Code demande adaptation
