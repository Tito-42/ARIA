## What It Does
Goose est un agent IA local créé par Block (la société derrière Square et Cash App). Construit en Rust (58%) et TypeScript (34%), il va au-delà des simples suggestions pour exécuter de façon autonome des workflows de développement complets. App desktop + CLI, extensible via MCP, 125 releases.

## Key Features
- **Execution autonome** de workflows dev complets
- **Multi-LLM** : sélection configurable de modèles
- **MCP servers** : extensibilité via protocole standard
- **Desktop + CLI** : deux interfaces
- **Rust-based** : 58.2% Rust, performance native
- **125 releases**, v1.29.0 (mars 2026)
