## What It Does
OpenHands (anciennement OpenDevin) est une plateforme de développement autonome alimentée par l'IA. Elle fournit un Agent SDK composable, une CLI, une GUI locale avec React, et un déploiement cloud avec intégrations Slack/Jira/Linear. Score SWE-Bench de 77.6 - parmi les meilleurs. Utilisé par TikTok, VMware, Amazon, Netflix, Apple, NVIDIA, Google.

## Key Features
- **Software Agent SDK** : bibliothèque Python composable
- **Multi-interface** : CLI, GUI locale (React), cloud
- **Multi-LLM** : Claude, GPT, et autres
- **Intégrations** : Slack, Jira, Linear
- **Enterprise** : self-hosted via Kubernetes, RBAC, multi-user
- **SWE-Bench 77.6** : top performer mondial
- **v1.6.0** (mars 2026) : développement très actif
