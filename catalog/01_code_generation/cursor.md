## What It Does
Cursor est un éditeur de code AI-first construit comme fork de VS Code. Il combine autocomplete intelligent (Tab), chat contextuel, mode Agent autonome multi-fichiers, et Composer pour génération/édition simultanée sur plusieurs fichiers. Support multi-model (Claude, GPT-4, Gemini).

## Key Features
- **Tab** : autocomplete intelligent multi-lignes
- **Agent mode** : coding autonome multi-fichiers
- **Composer** : génération/édition simultanée multi-fichiers
- **Chat** : contextuel avec @codebase, @file, @folder
- **Cloud agents** : tâches longues en arrière-plan
- **VS Code compatible** : extensions, settings, raccourcis
- **Multi-model** : Claude, GPT-4, Gemini, etc.
- **30+ partner plugins**

## Strengths
- Expérience IDE complète (pas juste un terminal)
- Transition transparente depuis VS Code
- Multi-model : choisir le meilleur modèle par tâche
- Agent mode très puissant

## Limitations
- Propriétaire (pas open source)
- Code envoyé au cloud (privacy concerns)
- $20-40/mois + coûts API
- Fork VS Code = toujours un pas derrière les updates VS Code
