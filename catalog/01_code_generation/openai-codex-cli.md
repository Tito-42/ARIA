## What It Does
Codex CLI est l'agent terminal d'OpenAI, construit principalement en Rust (94.7%) pour des performances maximales. Il s'exécute localement et s'intègre avec les abonnements ChatGPT. Support VS Code, Cursor et Windsurf.

## Key Features
- **Rust-based** (94.7%) pour performance native
- **Local-first** : exécution locale
- **Multi-plateforme** : macOS, Linux
- **ChatGPT integration** : Plus, Pro, Team, Enterprise
- **IDE support** : VS Code, Cursor, Windsurf
- **Développement actif** : 4,981 commits, v0.118.0 (mars 2026)
