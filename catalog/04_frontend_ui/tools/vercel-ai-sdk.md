# Vercel AI SDK (Frontend UI Reference)

> Cross-reference: The Vercel AI SDK is fully cataloged in [02_ai_agents/frameworks/vercel-ai-sdk.md](../../02_ai_agents/frameworks/vercel-ai-sdk.md).

## Relevance to Frontend/UI
The Vercel AI SDK is the leading TypeScript toolkit for building AI-powered user interfaces. While cataloged under AI Agents for its ToolLoopAgent capabilities, it is equally important for Frontend/UI because of:

- **UI hooks**: `useChat`, `useCompletion`, `useAssistant` for React, Svelte, Vue, Nuxt
- **Streaming UI**: First-class support for streaming AI responses in UIs
- **Generative UI**: AI-generated React Server Components
- **Multi-provider**: Unified API for OpenAI, Anthropic, Google in frontend apps
