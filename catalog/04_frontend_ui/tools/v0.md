## What It Does
v0 is Vercel's AI-powered generative UI platform that creates React and Next.js components, pages, and full-stack applications from natural language prompts. Users describe what they want in a chat interface, and v0 generates production-ready code using shadcn/ui, Tailwind CSS, and React Server Components. It supports iterative refinement -- users can modify generated components through follow-up prompts, edit code directly, and deploy to Vercel with one click.

v0 has become the de facto tool for rapid React/Next.js prototyping. It generates not just UI components but also API routes, database schemas (with Drizzle ORM), and authentication flows. The generated code is fully ownable -- no vendor lock-in beyond the Vercel ecosystem.

## Key Features
- **Chat-based generation**: Describe UI in natural language, get React code
- **shadcn/ui + Tailwind**: Generates accessible, styled components out of the box
- **Full-stack support**: API routes, database schemas, auth flows
- **Iterative refinement**: Modify components through follow-up prompts
- **Live preview**: Real-time preview of generated components in browser
- **One-click deploy**: Deploy to Vercel directly from v0
- **Code export**: Copy/paste or fork into your own project
- **Multi-file generation**: Creates complete page layouts and app structures
- **Image/screenshot input**: Upload designs to generate matching code
- **Next.js App Router**: Native support for modern Next.js patterns

## Limitations
- React/Next.js only (no Vue, Angular, Svelte)
- Proprietary and cloud-only (no self-hosting)
- Free tier has limited generations per month
- Best results require knowledge of React to guide prompts
- Vercel ecosystem dependency for deployment features
