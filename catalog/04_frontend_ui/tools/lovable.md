## What It Does
Lovable (rebranded from GPT Engineer in late 2024) is an AI-powered full-stack application builder. Users describe applications in natural language and Lovable generates complete, deployable web applications with both frontend (React + Tailwind + shadcn/ui) and backend (Supabase for database, auth, and storage). It targets non-technical founders, product managers, and developers who want to rapidly build and iterate on full-stack applications.

The platform differentiates itself through deep Supabase integration (managed database, auth, realtime, and file storage), GitHub sync for version control, and a visual editing experience. Lovable positions itself as "the last piece of the puzzle for prototyping" -- going from idea to deployed, functional app in hours rather than weeks.

## Key Features
- **Natural language to full-stack app**: Describe what you want, get a complete application
- **Supabase integration**: Built-in database, auth, storage, and realtime
- **React + Tailwind + shadcn/ui**: Modern, accessible frontend stack
- **GitHub sync**: Automatic Git integration for version control
- **One-click deploy**: Deploy to Lovable hosting or custom domains
- **Visual editor**: Edit UI elements visually alongside AI prompts
- **Iterative development**: Refine through conversation
- **Database management**: Visual schema editor with Supabase
- **Auth flows**: Pre-built authentication (email, OAuth)
- **Custom domains**: Connect your own domain to deployed apps

## Strengths
- End-to-end full-stack generation (not just frontend)
- Deep Supabase integration for managed backend
- Excellent for non-technical users
- GitHub sync maintains code ownership
- Visual + AI editing combination

## Limitations
- Locked into React + Supabase stack
- Proprietary platform (no self-hosting)
- Complex business logic requires manual coding
- Pricing scales with usage
- Generated code can be verbose/over-engineered for simple apps
