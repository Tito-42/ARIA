# 04 - Frontend & UI Generation

> Outils IA pour générer des interfaces utilisateur, composants web et applications frontend

## Vue d'ensemble
Les outils de cette catégorie permettent de générer du code frontend (HTML, CSS, React, Vue, etc.) à partir de descriptions en langage naturel, de screenshots, ou de maquettes. Ils accélèrent considérablement le prototypage et le développement UI.

## Outils à Cataloguer
- [ ] v0 by Vercel - Génération de composants React/Next.js
- [ ] Bolt.new - Full-stack AI development in browser
- [ ] Bolt.diy - Version open source de Bolt
- [ ] Lovable (ex GPT Engineer) - AI app builder
- [ ] Replit Agent - AI coding assistant with deployment
- [ ] Create.xyz - AI app generation
- [ ] Screenshot-to-code - Convertir screenshots en code
- [ ] Galileo AI - UI design generation
- [ ] Uizard - Design to code
